## ⚙️ 阿里云后台配置 (至关重要)

安装脚本后，请务必完成以下两项设置，否则将无法正常连接：

### 第一步：开启防火墙端口
1.  登录 [阿里云控制台](https://www.aliyun.com/) -> 进入 **“轻量应用服务器”** 管理页面。
2.  在左侧菜单或列表中点击 **详情** -> **防火墙** -> **添加规则**。
3.  **协议**：选择 `TCP`（若需使用 WhatsApp/Telegram 语音通话，建议同时添加一条 `UDP` 规则）。
4.  **端口**：填入您在脚本安装过程中设置的端口（默认端口为 `54321`）。

### 第二步：配置住宅 IP 白名单
1.  登录您的住宅 IP 服务商（如 [ipipd.com](https://ipipd.com?ref=HSMBEEGM)）后台。
2.  在后台面板中找到 **“授权 IP”** 或 **“白名单管理”** 选项。
3.  将您 **阿里云服务器的公网 IP** 添加到该白名单中。*（注：如果不添加，落地代理将拒绝来自中转服务器的连接）*

## ✨ 脚本核心特性

* **全交互式操作**：支持在安装时自定义端口、IP 及账号密码。针对 Linux 环境优化，完美支持 `Backspace` 退格修改（需使用官方 `curl` 命令启动）。
* **性能深度优化**：部署过程中自动开启内核 **BBR 加速**，有效降低跨境网络延迟，提升传输稳定性。
* **UDP 转发支持**：支持全协议转发，完美适配 WhatsApp、Telegram 等社交软件的语音及视频通话。
* **小型化二维码**：针对终端显示窗口进行特殊优化。生成的二维码体积小巧、不占屏幕，支持手机扫码即连。

## 🆘 常见问题排查

| 报错现象 | 原因及解决方法 |
| :--- | :--- |
| **404 Not Found** | 请检查安装命令是否完整，并确保您的服务器网络可以正常访问 GitHub。 |
| **Permission denied** | 权限不足。请切换至 `root` 用户运行，或在命令最前面添加 `sudo`。 |
| **\r: command not found** | 脚本检测到 Windows 换行符。请直接使用上方的 `curl` 命令在线安装，**切勿**手动下载后再上传文件。 |
| **连接超时 / 无法上网** | 1. 检查阿里云防火墙端口是否已开放；<br>2. 检查住宅 IP 服务商侧是否已将服务器 IP 加入白名单。 |
