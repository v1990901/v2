# V2Ray 美国静态住宅 IP 中转一键脚本

本项目专为跨境电商（TikTok/FB/Amazon）及外贸业务设计，通过阿里云轻量服务器中转，实现本地终端安全、稳定地接入美国原生住宅网络。

---

### 🚀 快速开始

在 Ubuntu 22.04 终端中执行以下命令即可启动交互式安装：

```bash
bash <(curl -sL [https://raw.githubusercontent.com/v1990901/v2/main/v2ray_relay.sh](https://raw.githubusercontent.com/v1990901/v2/main/v2ray_relay.sh))

```

⚠️ 阿里云配置注意事项（必看）
为了确保脚本安装后能够正常连接，请务必在阿里云后台完成以下操作：

放行防火墙端口：

登录阿里云控制台 -> 轻量应用服务器 -> 详情 -> 防火墙。

点击“添加规则”，协议选择 TCP，端口填入你在脚本中设置的监听端口（如 54321）。

如果你需要使用 WhatsApp/Telegram 语音，请同时放行该端口的 UDP 协议。

住宅 IP 授权白名单：

登录你的住宅 IP 提供商后台。

找到“授权 IP 管理”或“IP 白名单”，将阿里云服务器的公网 IP 添加进去。否则住宅 IP 会拒绝阿里云的连接请求。

✨ 功能亮点
全交互式体验：支持输入端口、IP、账号密码，支持 Backspace 退格删除纠错。

性能深度优化：自动开启内核 BBR 加速，显著提升跨境访问速度。

高兼容性：修复了 Windows 环境编辑导致的换行符乱码问题。

便捷输出：安装成功后直接在终端渲染小型化二维码，手机扫码秒连。

💡 常见问题排查
提示 404 或 \r: command not found：请确保通过上方的 curl 命令在线安装，不要在 Windows 记事本里手动复制脚本。

连接超时：90% 的原因是阿里云防火墙没开，或者住宅 IP 侧没加白名单。

二维码太大：请尝试按 Ctrl + - 缩小终端字体。
