# V2Ray 美国住宅 IP 中转脚本

### 🚀 一键安装命令
在 Ubuntu 22.04 终端执行：
\`\`\`bash
bash <(curl -sL https://raw.githubusercontent.com/v1990901/v2/main/v2ray_relay.sh)
\`\`\`

### 🛠️ 功能特点
- **全交互式**：支持输入端口、美国静态 IP、账号密码，支持退格删除。
- **性能优化**：自动开启内核 BBR 加速，支持 UDP 转发（适合社交软件通话）。
- **小型化二维码**：适配各种终端窗口，扫码即用。

### ⚠️ 注意事项
1. 请务必在阿里云控制台放行你设置的 **入站端口**。
2. 确保你的美国住宅 IP 已授权阿里云服务器的公网 IP。
